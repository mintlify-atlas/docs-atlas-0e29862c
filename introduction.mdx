react-declarative is a TypeScript React library that lets you describe your entire UI — forms, data grids, dashboards, and app shells — as a plain JSON/TypeScript schema. The library renders MUI components, handles state management, validation, and responsive 12-column grid layout automatically, so you write configuration instead of boilerplate.

## Key capabilities

- **Declarative schema** — describe fields, layouts, and validation rules as a typed `TypedField[]` array; the library does the rest
- **Automatic state management** — connect a `handler` function or object; `One` fetches, tracks, and emits `onChange` automatically
- **Responsive 12-column grid** — specify `phoneColumns`, `tabletColumns`, and `desktopColumns` per field group
- **Rich validation** — `isInvalid`, `isDisabled`, `isVisible`, and `isReadonly` callbacks receive live form state
- **Slot factory** — swap any default MUI field renderer for your own component without changing the schema
- **Async hooks** — `useSinglerunAction`, `useQueuedAction`, `useAsyncProgress` for safe concurrent UI interactions
- **MVVM** — `useCollection`, `useModel`, `useEntity` for ref-managed reactive data collections
- **Built-in router** — `Switch` with guards, prefetch, unload, and redirect support
- **Dependency injection** — `provide`, `inject`, `createServiceManager` for IoC-style service wiring
