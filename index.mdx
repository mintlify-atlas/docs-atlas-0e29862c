---
title: "react-declarative: JSON-driven React UI framework"
description: "react-declarative turns TypeScript schemas into MUI forms, data grids, and app shells with automatic state, validation, and responsive layout."
---

react-declarative is a TypeScript React library for building forms, data grids, dashboards, and application shells from a declarative JSON schema. MUI components are rendered automatically; state management, validation, and responsive layout are handled by the library.
